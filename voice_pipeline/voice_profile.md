# Mochi 破帽男声线设定

## 核心感觉

Mochi 的声音不是成熟男歌手，也不是标准年轻男声。他更像一个小只、嘴硬、丧丧的卡通角色，声音偏童音，轻、软、细一点，但不能太甜。情绪藏着，不会大哭大喊。

## 声线关键词

```text
小只男孩感
一点童音
偏高但不尖
柔软
轻微鼻音
一点点哑
慵懒
克制
悲伤
小声
不爆发
不模仿真实歌手
```

## 唱歌方式

```text
像一只小小的破帽 Mochi 在雨夜房间里抱着小吉他唱给自己听。
音量不大，尾音短一点、软一点，有一点孩子气的委屈。
副歌也不要太用力，情绪压住，像“我没事”的那种嘴硬难过。
```

## 不要的方向

```text
不要像明星歌手
不要厚重男低音
不要普通成年男声
不要太成熟
不要摇滚嘶吼
不要太甜美
不要播音腔
不要真实小孩声音
不要二次元女声
```

## 英文提示词

```text
small childlike male cartoon voice, soft slightly high tone, gentle nasal color, a little raspy, tired and restrained, melancholic, cute but lonely, intimate acoustic indie style, not powerful, not dramatic, does not imitate any real singer or character
```

## 中文提示词

```text
小只男孩感的卡通声线，有一点童音，偏高但不尖，柔软，轻微鼻音和一点点哑，慵懒克制，悲伤但不夸张，像在雨夜抱着小吉他小声唱歌。不要模仿任何真实歌手或具体角色。
```
